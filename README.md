# Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript. Live at [portfolio-two-jade-63.vercel.app](https://portfolio-two-jade-63.vercel.app).

## Stack

- Vanilla HTML / CSS / JavaScript — no framework
- Deployed on Vercel (auto-deploys on push to main)

## Structure

```
portfolio/
├── index.html      # Main page
├── css/            # Stylesheets
├── js/             # Scripts
└── Resume.pdf      # CV
```

## Running locally

Open `index.html` directly in a browser — no build step required.
