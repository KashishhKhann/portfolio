GraphRAG Clinical Decision Support

NLP / Knowledge Graph / RAG · 2025–2026

  • End-to-end five-stage pipeline over 50,000+ MIMIC-IV clinical notes: section-aware chunking (7 note types, type-specific token targets) → BioClinicalBERT embeddings (768-dim, CLS pooling) → FAISS indexing → Neo4j knowledge graph → hybrid retrieval.
  • Hybrid ranking combining dense FAISS retrieval (85% weight), BM25 lexical reranking (25%), KG concept overlap with IDF-style frequency penalties (15%), and section-bias inference — catching exact drug names and ICD codes that dense models miss.
  • Child-span refinement re-segments parent chunks into sub-spans and re-ranks by cosine similarity at query time, achieving sentence-level precision without storage overhead. Within-admission scoping prevents cross-patient data leakage.
  • Fine-tuned BioClinicalBERT for clinical NER; entities enriched with ICD-10/ICD-9 codes and stored as Neo4j concept nodes. Evaluated across 50 tasks / 107 cases with ROUGE-L, BERTScore F1, hit@k, and precision@k.
